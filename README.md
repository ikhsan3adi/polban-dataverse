# Polban DataVerse

> Polban Dataverse is a collaborative initiative
to consolidate data from various sources within the Bandung State Polytechnic
into a single interactive visual portal.

## Overview

This superproject repository contains:

- All DataVerse services as git submodules:
  - [**Polban DataCore**](https://github.com/muhamadSyahid/polban-datacore)
  - [**Polban DataHub**](https://github.com/ErsyaHasby/polban-datahub)
  - [**Polban DataView**](https://github.com/ihsan-ramadhan/polban-dataview)
- Dockerfile for building all services
- Docker Compose configuration for easy deployment

## How to Use

### Start

Run the script:

```sh
./run.sh
```

### Stop

Run:

```sh
docker compose down
```
